# MyPrefix
a useful prefix plugin for Genisys core

本插件尚未完善
This plugin is not done now.

指令:
- /prefix help [指令幫助]
- /prefix np <稱號名> <花費> [(op權限) 創建一個新的稱號]
- /prefix set <稱號ID> [更換你自己的稱號]
- /prefix mylist [查看你自己的所有稱號]

功能:
- 目前已有基礎功能

即將更新:
- 更多指令 與 功能

API使用方式:
